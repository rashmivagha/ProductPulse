# ProductPulse

**Interactive Sales Dashboard for Superstore Dataset**

---

## Project Overview

ProductPulse is an interactive Tableau dashboard that analyzes sales performance for a retail dataset (Superstore). The goal is to provide insights into overall sales, profits, orders, top product categories, and top-selling products, with easy-to-use filters for dynamic analysis.
---

## Features & Visualizations

1. **Key Metrics (KPI Row)**
   - Sales, Profit, Orders
   - Quick overview of business performance
   - Numbers formatted as currency and easy to read

2. **Monthly Sales Trend**
   - Line chart showing sales over time
   - Interactive **Region filter** for dynamic analysis

3. **Category Breakdown**
   - Bar chart of sales by product category
   - Filtered by Region for interactivity
   - Shows total sales per category clearly

4. **Top Products**
   - Bar chart of top-selling products
   - Shows sales value and optional color coding for profit
   - Scrollable for large product lists

---

## Interactivity

- **Region filter:** Applies to all charts simultaneously
- Users can select US, Canada, or both to dynamically update all visualizations

---
## Dataset

- **Source:** Tableau Sample Superstore dataset  
- Includes orders, sales, profit, product categories, and regions  
- Used for analysis of sales trends and product performance

---

## Key Insights

- Quickly identify high-performing regions and products  
- Monitor monthly sales trends for business decisions  
- Analyze category-level performance and profitability

## Contribution Guide

- How to Contribute: Fork the repository and submit pull requests with clear descriptions.  
- Coding / Visualization Standards: Follow existing naming conventions for sheets, calculated fields, and filters. 
- Documentation: Update README and screenshots when adding or modifying features.
- Review: Ensure changes maintain dashboard performance and interactivity.

- Best Practices: Keep PRs focused and incremental. Avoid large, unrelated changes in a single submission to simplify review and maintenance.
